算法的基础和素养

- 题目
    - 需要把**单向**链表反转
- 迭代解法 O(n) 时间复杂度 O(1)空间复杂度，重点讲清除指针的变化过程。
    - 双指针 pre cur next
    每到一个结点， 将其next 指针指向前一个结点pre,然后prev 和 cur 依次前移，实现链表方向的逐步翻转。
    最后返回prev 指针，指向新的头结点。

- 递归法
    假设子问题已经解决，
    1->2->3->null 翻转
    head 1
    之前已经完成的next 指向 head 
    更好理解的 

    - 先要走到最后一个结点
    - 然后从后往前翻转（假设已经搞定了） 回溯
